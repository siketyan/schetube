# schetube
A CLI tool to fetch upcoming live streams from a YouTube channel.

## Installation
```
$ git clone https://github.com/siketyan/schetube.git
$ cd schetube
$ cargo build --release
$ ln -s "$(pwd)/target/release/schetube" /usr/local/bin/schetube
```

## Usage
```
$ schetube [Channel ID]
```
